# Jak używać Ikon

Aby użyć ikony należy pobrać paczkę i załączyć plik: style.css
do index.html. Następnie wykorzystac klasy zawarte w style.css.
Przykład użycia:

<Source
  language="css"
  format={true}
  code={`
    <span class="icon-Beer" />
  `}
/>
