# Zastosowanie

Mniejsze przestrzenie (5px, 10px) stosuje się w celu rozdzielenia mniejszych elementów
np.: gdy chcemy oddzielić tekst albo tokeny. Można stosować również w przypadku
Oddzielenia tekstu od kontenera
Większe przestrzenie (20px, 25px) używamy np.: w przypadku przycisków, modali,
często przy powiększeniu niewielkiego kontenera w celu poprawienia czytelności.
Największe przestrzenie (30px, 40px) stosuje sie gdy chcemy rozdzielić sekcję. Często uźywa się
się w przypadku landingów, aby urozmaicić stronę.
