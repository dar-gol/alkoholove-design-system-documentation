# Wstęp

W tej sekcji porzedstawione zostały wszystkie dostępne kolory zawarte w
palecie kolorów Alkoholove. Wyjasnione zostało kiedy stosować dany kolor a
kiedy lepszym pomysłem jest zastosowanie innego. Wszystkie kolory dostępne są
w obiekcie `theme.palette`

<a name="colors_kind" />

# Rodzaje kolorow

- **Primary** - należy używać w większości elementów, które chcemy uwydatnić.
  Na każdym widoku powinnien znajdować się ten widok, ponieważ jest to jednocześnie
  kolor marki. Stosowany jest w przyciskach Call To Action, logo, aktywne elementy.
- **Secondary** - należy stosować w elementach drugorzędnych oraz w elementach w
  których wiemy że użytkownik zwróci uwagę (wtedy nie trzeba stosować Primary kolor
  ze względu na to iż wiemy ze użytkownik skupi swoją uwagę na danym elemencie)
  Kolor stosowany również w linkach w tekście.
- **Grey** - kolor stosowany do tekstów, tła, obramowań, cieni. Stosowany w większości
  elementach.
- **Red** - Kolor informujący o niepowodzeniu, lub w celu poinformowania użytkownika
  o krytycznych elementach. Kolor powinnien być bardzo rzadko stosowany, właściwie
  w jednym miejscu na danym widoku.
- **Yellow** - kolor ostrzegawczy, stosowany w celu poinformowania użytkownika
  o możliwych nieporządanych skutkach jakiegoś działania. Ten kolor upewnia nas,
  że użytkownik przeczyta daną informację. Do informacji która jest mniej istotna
  stosuj kolor Secondary. Tego koloru nie powinno być za dużo na jednym widoku.
- **Green** - kolor informującu o powodzeniu danego działania jak i jako znacznik
  informujący o nowościach itd. Wskazuje pozytywne aspekty.

<a name="colors_value" />

# Wartości kolorów

Poniżej przedstawione zostały wartości powyżej podanych rodzajów kolorów.
Wartości są z zakresu od 0 do 100. Czym większa wartość tym większa intensywność.
Wartości 100 są bardzo rzadko stosowane na przykład do elementów aktywnych.
Każda wartość posiada motyw Light i Dark.

## Obiekt Theme

Powyżej został stworzony obiekt Colors zawierający wszystkie potrzebne kolory.
Teraz należy dodać obiekt Colors do theme, aby kolory były dostępne globalnie
w calym styled-components. Tutaj przedstawię po krótce jak to utworzyć. Nie będę
pokazywał jak stworzyć providera. To możecie znaleźć na stronie styled-components.com.
W pliku theme.ts tworzymy:

<Source
  language="js"
  format={true}
  code={`
    import Colors from "./colors.styled";
    const setMode = (mode) => {
    if (["light", "dark"].includes(mode)) return mode;
    return "light";
    };
    export const createTheme = (mode = "light") => {
      const validMode = setMode(mode);
      return {
        palette: Colors[validMode],
      };
    };
    export default createTheme;
`}
/>

Następnie wartość z funkcji createTheme dodajemy do providera.
Na przykład tak jak w przykładzie poniżej

<Source
  language="jsx"
  format={true}
  code={`
    ...
    const theme = createTheme(cookie?.mode, cookie?.isHighContrast === "true");
    ...
    <ThemeProvider theme={theme}>
    ...
    </ThemeProvider>
    ...
`}
/>
